# Frontend — Compass Skills Graph

Vite + React + TypeScript. See the **[project README](../README.md)** for features (drag nodes, click for skill details, API shape) and how to run the full stack.

```bash
npm install
npm run dev
```

Optional: `VITE_API_BASE_URL` to your FastAPI origin (e.g. `http://127.0.0.1:8000`).
